# Internship-task-13---BI-Dashboard-Storytelling-KPI-Report

## Objective
To build an executive-ready interactive KPI dashboard that provides insights into sales performance, profitability, regional performance, and top-performing products using the Global Superstore dataset.

## Dataset
- Global Superstore (superstore.csv)

## Tool Used
- Power BI Desktop

## Dashboard Components

## KPI Cards
Displayed at the top of the dashboard:
- Total Sales
- Total Profit
- Profit Margin %

## Sales Trend Analysis
- Line chart showing Sales over Time (Month-Year)
- Helps identify growth trends and seasonality

## Category Performance
- Clustered Bar Chart
- Sales by Category
- Sorted in descending order

## Regional Performance
- Column Chart
- Sales by Region
- Identifies best-performing regions

## Top 10 Products
- Table visual
- Filtered using Top N (10) by Total Sales
- Displays Product Name, Sales, and Profit

## Interactive Slicers
The following slicers were added for dynamic filtering:
- Region
- Category
- Order Date (Between range)
All visuals update automatically based on slicer selections.

## Final Outcome
An executive-level interactive dashboard capable of supporting strategic business decisions through clear KPIs, trend analysis, regional comparison, and data storytelling.
